# 50 Startups - Profit Prediction

## Objective
Predict profit using R&D Spend, Administration, Marketing Spend, and State.

## Approach
- Performed EDA
- Applied Multiple Linear Regression
- Tested transformations
- Selected best model based on R²

## Results
- Best Model: Linear Regression
- R² Score: ~0.95
- Key Feature: R&D Spend

- Notebook (.ipynb + .html)
