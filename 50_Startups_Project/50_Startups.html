# Choosing the best model
Multiple linear regression was used to predict profit based on R&D Spend, Administration, Marketing Spend, and State. Exploratory Data Analysis showed that R&D Spend has the highest correlation with profit, making it the most significant predictor. The model achieved an R² value of 0.95, indicating excellent predictive performance. Transformations such as log transformations were applied, but they did not significantly improve the model. Therefore, the linear regression model is selected as the final model.
